# SpaceShooter

### Introduction

Ce jeu est un space shooter, réaliser par Paul Kehlhoffner et Aymeric Martin.

Le joueur possède se retrouve dans l'espace avec son vaisseau spacial, et doit détruire le vaisseau mère sur sa route.
Malheureusement, ce vaisseau mère appelle des vaisseaux ennemis pour se protéger, et vous attaquer en même temps.
